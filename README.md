# ThaurosiaCraft Public Library
This Repository is meant to store all of the official Documents of ThaurosiaCraft such as rules, Descriptions and any other Text, mostly written out as markdown files.

## Why did we choose GitHub?
- It serves as one place where we can store all of the important texts.
- Easy overseeing all of the files and version controll, easy blame and changes review.
- Transparency towards our Playerbase.
